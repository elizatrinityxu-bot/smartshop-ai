# SmartShop AI

A Django-based AI-powered e-commerce platform featuring:

- Personalized product recommendations
- NLP-based smart search
- Virtual shopping assistant chatbot
- AI-generated review summaries
- Dynamic product descriptions

## Tech Stack
- Django / Django REST Framework
- MySQL
- OpenAI API
- Bootstrap 5

## Key Features
- AI-powered content-based recommendations
- Cached AI summaries for cost efficiency
- Context-aware chatbot with product understanding
